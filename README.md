# txLogin

Standalone FiveM resource that bridges txAdmin authentication with a togglable "Duty" status.

📖 **Full documentation:** https://jhag01.github.io/fivem-docs/txlogin/

## Requirements

- txAdmin (bundled with FXServer)
- `ox_lib` — optional, only needed for `Notify`/`Logger` set to `'ox'`

## Quick start

1. Clone into `resources/`
2. `ensure txLogin` in `server.cfg`
3. See the [docs](https://jhag01.github.io/fivem-docs/txlogin/) for configuration, the required txAdmin menu edit, exports, and everything else.
